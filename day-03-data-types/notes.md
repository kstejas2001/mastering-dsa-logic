# Day 03 Notes

## Topic: Data Types

### What I learned today
A computer stores different kinds of data in different ways.

Not all values are the same.
Some values are numbers.
Some are text.
Some are true or false.

### What is a data type
A data type tells the computer what kind of value a variable has.

### Main data types I learned today
- **int** → whole numbers like 5, 10, 100
- **float** → decimal numbers like 3.5, 9.8
- **str / string** → text like "hello"
- **bool** → true or false values

### Why data types are important
Because the computer needs to know:
- how to store the value
- how to use the value
- whether addition, comparison, or text work should happen

### Example thinking
- `age = 20` → number
- `price = 99.5` → decimal number
- `name = "Crazy"` → text
- `is_student = True` → true/false

### Important idea
If we use the wrong data type, the program may give an error or wrong result.

### My understanding
A variable is like a box.
A data type tells what kind of thing is inside that box.

# Day 05 Notes

## Topic: Operators and Expressions

### What I learned today
A computer can do work with values using operators.

Operators help us:
- add numbers
- subtract numbers
- multiply numbers
- divide numbers
- compare values
- make decisions later

### What is an operator
An operator is a symbol that tells the computer what action to perform.

### Common operators
- `+` → addition
- `-` → subtraction
- `*` → multiplication
- `/` → division
- `%` → remainder
- `==` → checks if two values are equal
- `!=` → checks if two values are not equal
- `>` → greater than
- `<` → less than
- `>=` → greater than or equal to
- `<=` → less than or equal to

### What is an expression
An expression is a combination of values and operators.

Example:
- `5 + 3`
- `10 - 2`
- `a + b`

### Why this is important
Operators are used in almost every program.
Without operators, we cannot calculate or compare values.

### My understanding
- Values are the data
- Operators do the work
- Expressions are the full calculations

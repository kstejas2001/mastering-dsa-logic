# Day 04 Notes

## Topic: Type Conversion

### What I learned today
Sometimes a value is stored as one type, but we need to use it as another type.

Example:
- text to number
- number to text
- whole number to decimal

### What is type conversion
Type conversion means changing the data type of a value.

### Why it is needed
A computer follows rules for each data type.

If we try to add text and numbers directly, the program may give a wrong result or an error.

So we convert one type into another type when needed.

### Important words
- **Type**: the kind of data
- **Conversion**: changing from one type to another
- **Casting**: another word for conversion in some languages

### Common cases
- `"5"` → number `5`
- `5` → text `"5"`
- `10` → `10.0`

### Why this matters
In real programs, user input often comes as text first.
Before calculation, we usually convert it into numbers.

### My understanding
A variable stores a value.
A data type tells what kind of value it is.
Type conversion changes that kind.

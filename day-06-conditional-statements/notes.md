# Day 06 Notes

## Topic: Conditional Statements

### What I learned today
A program can make decisions.

It can check a condition and choose what to do next.

### What is a condition
A condition is something that is either `true` or `false`.

Example:
- `10 > 5` → true
- `3 > 5` → false

### Why conditions are important
Without conditions, every program would do only one fixed action.

With conditions, the program can behave differently in different situations.

### Common words
- **if** → do something only when the condition is true
- **else** → do something when the condition is false
- **elif** → check another condition in between

### Simple idea
If the condition is true, one block of code runs.
If the condition is false, another block may run.

### Example thinking
- If it is raining, take an umbrella.
- Otherwise, do not take it.

### My understanding
Conditions help the computer choose between different paths.

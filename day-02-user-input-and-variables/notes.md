## Topic: User Input & Variables

### What I learned today
A program can take input from the user.
Instead of fixed values, we can ask the user to enter data.

### Important words
- **Input**: data given by the user
- **Output**: result shown by the program
- **Variable**: stores user input

- We used fixed values (a = 5)

Today:

### Example thinking

- Show result

### Important difference
- Before → values were hardcoded
- Now → values come from user
